Help

Welcome to the Actvis help page. Here you can find information on how to use our data visualization tool and how to export data in various formats. Additionally, we have provided answers to some frequently asked questions below.

Frequently Asked Questions

  • How often is the data updated?

    The data is updated daily, and you can see the last updated date and time on the dashboard.

  • Can I filter the data by category or year?

    Yes, you can filter the data by category or year using the filter options on the dashboard.

  • How many nominations does each actor have?

    You can view the number of nominations for each actor on the actor's profile page. Simply click on the actor's name to access their profile.
